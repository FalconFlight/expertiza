Package Include:
1. Design Document (PDF)
2. Proogram Source Code
3. Database dump file (pg_dump.sql)

Steps to deploy your program
1. Unzip expertiza.zip to the folder at your choice
2. Open up project, run the latest database migration
3. Use MySql console to restore the database from dump file pg_dump.sql, the name of the database to be restored should be pg_development
4. Start the server, the environment should be development
5. type the correct URL in the browser, then you are ready to go
